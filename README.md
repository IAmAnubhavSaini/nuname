# nuname
node uname

Usage:

```
npm install nuname // for installing this to your project.

// or directly in node.

var nuname = require('nuname');

console.log(nuname('r')); // for release
console.log(nuname('s')); // for type
console.log(nuname('t')); // for hostname

```
