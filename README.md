# nuname
node uname

Usage:

```
npm install nuname // for installing this to your project.

// or directly in node.

var nuname = require('nuname');

console.log(nuname('r')); // for system release
console.log(nuname('s')); // for operating system name
console.log(nuname('n')); // for nodename

```
